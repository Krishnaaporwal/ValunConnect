Build a full-stack web application called **"VolunteerConnect"** using modern technologies (Next.js + TypeScript + Tailwind CSS + Node.js + MongoDB or Supabase).

The platform should support **two roles: Volunteer and Organizer**, with a highly user-friendly UI and scalable backend.

---

Implement secure authentication using JWT or Supabase Auth.

### Roles:

1. Volunteer
2. Organizer

* Name
* Email
* Password
* Skills (editable later)
* Interests
* Location

### Organizer Signup Fields:

* Organization Name
* Email
* Password
* Location
* Organization Type:

  * NGO
  * Government
  * Private

#### 🧠 AI-Based Recommendation System:

* Recommend events based on:

  * Skills
  * Interests
  * Location

#### 🔢 Matching Percentage System (IMPORTANT):

* Show "Match %" between volunteer profile and event requirements

## 🧠 7. INTELLIGENT MATCHING ENGINE

Build a function:

matchScore = (
skillsMatch * 0.5 +
locationMatch * 0.2 +
interestMatch * 0.2 +
availabilityMatch * 0.1
)

Return:

* Match %
* Tag:

  * High Match (75%+)
  * Medium (40–75%)
  * Low (<40%)
